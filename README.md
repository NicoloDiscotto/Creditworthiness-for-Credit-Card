# Creditworthiness for credit card issuance

## Project Description

Building a predictive model to assess a customer's creditworthiness, in order to support the decision on whether to approve a credit card application.

## Objective

- Exploratory Analysis (EDA)
- Handling of missing and anomalous data
- Training classification models (Logistic Regression, Random Forest, etc.)
- Performance evaluation (Accuracy, ROC AUC, Confusion Matrix)
