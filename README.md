# Credit Risk Prediction

## Project Description

Building a predictive model to assess a customer's creditworthiness, in order to support the decision on whether to approve a credit card application.

## Objective
Build a binary classification model that can:
- Predicting whether a customer will default
- Support more accurate credit decisions
- Reduce financial risk

## Dataset
The dataset contains customer financial and behavioral information, including:
- Demographic data (age, gender, marital status, education)
- Credit information (amt_income)
- Target: 1 = not reliable, 0 = reliable
  
## Library
- `scikit-learn`
- `numpy`
- `matplotlib`
- `pandas`
- `seaborn`

## Approach
1. Data Preprocessing
- Data Cleaning
- Missing Value Handling
- Categorical Variable Encoding
- Feature Scaling

2. Exploratory Data Analysis (EDA)
- Distribution Analysis
- Study of correlations between variables
- Identification of risk patterns

3. Feature Engineering
- Creating new features
- Selecting the most relevant variables

4. Modeling
- Logistic Regression
- Random Forest
- Decision Tree

5. Rating
Key metrics:
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
